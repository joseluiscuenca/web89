O R G A N I C   :   D A T A   [ 2 0 2 1 ]

Organic::DATA es un sistema interactivo para la interpretación audiovisual que combina de forma dinámica las visualizaciones y el sonido en tiempo real. En este Proyecto lo musical y lo visual son resultado del procesamiento de imágenes por un algoritmo diseñado que busca obtener, almacenar, clasificar y analizar la información biométrica de imágenes obtenidas del repositorio abierto C-MS-Celeb, estas imágenes sirvieron como datos de entrenamiento para ajustar parámetros del algoritmo, entrenar nuestra red neuronal y que funcione como regla de entrada a las imágenes que se interpretarán en tiempo real.

O R G A N I C   :   D A T A   [ 2 0 2 1 ]

Organic::DATA is an interactive system for audiovisual interpretation that dynamically combines visualizations and sound in real time. In this project, the musical and the visual are the result of image processing by an algorithm designed to obtain, store, classify and analyze biometric information from images obtained from the open repository C-MS-Celeb, these images served as training data to adjust parameters of the algorithm, train our neural network and work as an input rule to the images to be interpreted in real time.
